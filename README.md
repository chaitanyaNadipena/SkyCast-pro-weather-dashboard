

# 🌤️ SkyCast Pro - Weather Dashboard

A modern, responsive, and interactive weather dashboard built using **HTML, CSS, and JavaScript**. SkyCast Pro provides real-time weather information, location-based insights, and an elegant glassmorphism interface powered by the **OpenWeather API** and **Leaflet Maps**.

## ✨ Features

- 🌍 Search weather by city name
- 📍 Interactive map with location-based weather
- 🌡️ Real-time temperature updates
- ☁️ Current weather conditions
- 💧 Humidity information
- 🌬️ Wind speed
- 📈 Atmospheric pressure
- 👀 Visibility details
- 🌅 Sunrise time
- 📊 Weather Summary
- 📝 Today's Outlook
- ⚡ Quick city search buttons
- 🎨 Premium Glassmorphism UI
- 📱 Fully Responsive Design
- ⭐ Interactive rating section
